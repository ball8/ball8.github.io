# ball8.github.io

Stránka by sa mala správne zobrazovať na Mozille verzia 51.0.1 a vyššie, Google Chrome verzia 56.0.2924.87 a vyššie.
Pre fungovanie pluginu treba v súbore poem.html odkomentovať označený odstavec (jediný komentár v súbore).
Github blokuje plugin, pri nezakomentovanom použití filtra z tohot pluginu vloží obsah na stránku namiesto funkcie, ktorú má plugin vykonať, preto je zakomentovaný.
